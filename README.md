## RGB, CMY :: 색상테이블

<br/>

RGB:빛의 삼원색 뜻 빨간색(R), 초록색(G), 파란색(B)의 합성어

CMY:각각 시안(C), 마젠타(M), 옐로(Y), 검정(K)를 나타낸다

이에 대한 오프셋 인쇄에 쓰이는 4가지 색을 이용한 잉크체계
